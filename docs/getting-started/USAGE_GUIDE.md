## Current Product Shape

Orgumented is:
- a Windows-native Tauri desktop app
- an embedded Next.js operator UI
- a local NestJS semantic engine
- local `sf` keychain auth with `cci` as support tooling

The active product path is desktop-first, Ask-first, and proof-backed.
